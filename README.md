##제목: 맞춤거래기록부

테이블 5개까지 거래기록부

## 개발동기

현재 부모님이 자영업을 하심. 
거래일지를 엑셀로 문서화하심.
하지만 부모님 연세가 있으신지라 엑셀의 많은 기능을 활용 못함. 
그래서 부모님을 맞춤으로 필요한 기능만 넣고, 보기 편한 거래기록부를 만드는 것을 생각하게 됨.
